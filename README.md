# GIZ Brand Template

A simple static website template with GIZ branding for reference lists.

## Files

- `index.html` - Main HTML template
- `styles.css` - GIZ-branded stylesheet

## Brand Colors

- **GIZ Red**: #E20074 (primary accent color)
- **GIZ Red Dark**: #B8005C (hover states)
- **Black**: #000000 (text)
- **Gray**: #333333, #666666 (secondary text)
- **White**: #FFFFFF (background)

## Usage

1. Edit `index.html` to add your reference lists
2. Customize the styles in `styles.css` as needed
3. The template is responsive and works on mobile devices

## Structure

The template includes:
- Header with GIZ logo
- Main content area for reference lists
- Footer with GIZ attribution
- Clean, professional styling matching GIZ brand guidelines
